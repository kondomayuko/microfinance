# Contributing to the Simple Interest Calculator 🌟

First off, thank you for considering contributing to our project! It's people like you who help us build accessible, transparent financial tools to empower low-income individuals. 

We are currently transitioning our team to Git and GitHub, so whether you are a Git veteran or learning alongside us, we welcome your input, code, and feedback.

This document provides guidelines to ensure a smooth and collaborative contribution process.

## 📜 Code of Conduct

By participating in this project, you agree to abide by our [Code of Conduct](CODE_OF_CONDUCT.md). Please read it to understand the expectations for behavior in our community.

## 💡 How Can I Contribute?

There are many ways you can contribute to this project:

*   **Reporting Bugs:** If you find a bug in the calculation or the script, please open an issue.
*   **Suggesting Enhancements:** Have an idea to make the tool better? We'd love to hear it.
*   **Improving Documentation:** Fixing typos, clarifying instructions, or translating the README.
*   **Submitting Code:** Writing code to fix bugs or add new features.

## 🐛 Reporting Bugs & Suggesting Enhancements

Before creating a new issue, please check the [Issue Tracker](../../issues) to see if it has already been reported or suggested. 

When creating an issue, please include:
*   A clear and descriptive title.
*   A detailed description of the bug or enhancement.
*   Steps to reproduce the issue (if reporting a bug).
*   Your operating system and environment details.

## 🛠️ Contribution Workflow (Git & GitHub)

We use the standard "Fork-and-Pull" workflow. If you are new to Git, follow these steps to contribute your code:

### 1. Fork the Repository
Click the **Fork** button at the top right of this repository's page to create a copy of the project in your own GitHub account.
